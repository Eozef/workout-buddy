# Workout Buddy / MERN Stack
A Workout Journal Web application
## Installation and use

### Installation and use

Clone this repository:

```
git clone https://github.com/Eozef/workout-buddy.git

```

Install npm packages:

```
npm install
```

Start working:

```
npm start
```

Testing:

```
npm test a
```
